# Test Project

This is a sample project demonstrating all CacaoDocs doc types.

## Doc Types

- **function** - Regular Python functions
- **api** - REST API endpoints (auto-detected from FastAPI/Flask decorators)
- **class** - Python classes with methods
- **config** - Configuration and environment variables
- **event** - Webhooks and event handlers

## Getting Started

Install dependencies and run the server:

```bash
pip install fastapi uvicorn
uvicorn app:app --reload
```
